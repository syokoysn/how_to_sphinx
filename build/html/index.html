sphinxでwebサイト公開してみる¶

このサイトについて¶

このサイトは, Sphinxで Jpyter notebook を公開するまでを勉強できるサイトです.
以上です.

環境¶

  • Mac
注) Windows の環境では試していません.
コマンドを少し変えたらできると思います．

対象者¶

  • github のアカウントある
  • sphinx使ってみたい人
  • sphinxとgithub pagesで jupyter notebook を公開したい人
  • notebook を整理したい人

内容について¶

sphinxを使ってwebページを公開します.
githubのアカウントがあれば公開できます．
