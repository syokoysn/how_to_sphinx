詳しくはこちら
  1. プロジェクトのサイト

単純にリポジトリ作成したらいいです.
ここでは, この作り方のwebサイトを作るために how_to_sphinx というリポジトリを作成しました.

2. 必要なライブラリのインストール¶

必須なのは sphinx です.
下の二つは目的に応じてインストールしてください
  • sphinx
  • nbsphinx (jupyter notebookを公開したいなら)
インストール
pip でいけます
pip install Sphinx
pip install nbsphinx
とりあえず準備終わりです.
